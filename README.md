# Slytherin App

### React Native Application

## Getting Started

`git clone REPO`

`cd slytherin-app`

`yarn`

`cd ios`

`pod install`

from the root directory run app: `npx react-native run-ios`

## Libraries

- React Navigation v6
- Native Base
- Redux
